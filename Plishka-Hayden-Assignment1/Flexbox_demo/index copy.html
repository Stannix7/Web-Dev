In row direction, items flow left→right (main axis = horizontal) and cross axis is vertical. In column direction, items flow top→bottom (main axis = vertical) and cross axis is horizontal.

The container uses justify-content to space items along the main axis and align-items to align them on the cross axis. One item overrides with align-self.

Order (visual reordering)

First in HTML, order: 2
Second in HTML, order: -1
Third in HTML, order: 1

All items default to order: 0. Lower numbers appear earlier. Here: -1 → 1st, 1 → 2nd, 2 → 3rd.
